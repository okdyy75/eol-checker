## バージョンステータス計算ロジック

LTSがある場合

support - eol | maintenance
lts - support | active
release - lts | current

lts - eol     | maintenance
release - lts | current

LTSがない場合

support - eol     | maintenance
release - support | active

release - eol     | active

1. **EOL済み**
   - `eol`日が現在より過去なら、全期間を **eol**（赤）
2. **LTS日付あり**（`lts`が日付文字列）
   - `releaseDate → lts` は **current**（緑）
   - `lts → support` は **active**（青）（`support`がある場合）
   - `support → eol` は **maintenance**（グレー）（`support`がある場合）
   - `lts → eol` は **active**（青）（`support`がない場合、例: deno 2.5）
3. **LTSなし + supportあり**
   - `releaseDate → support` は **active**（青）
   - `support → eol` は **maintenance**（グレー）
4. **LTSなし + supportなし**
   - `releaseDate → eol` は **active**（青）
